# AL-Editor

> A visual graph editor based on X6 and Vue3.

## 预览

[在线预览](https://al-editor.vercel.app/editor)

## 本地运行(推荐使用pnpm)

```bash
# git clone https://github.com/Alexhueng/al-editor

# pnpm i

# pnpm dev
```
